# React-contacts

### Start Server
```
$ npm start
```

### Installing backend server
```
$ git clone https://github.com/udacity/reactnd-contacts-server.git
$ npm install
$ node server.js
```

$ npm install react-router-dom

$ npm install --save form-serialize
